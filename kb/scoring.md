# 比分预测

> Poisson + 让球联动 + D-Gate判型联动

## 核心公式

```
λ_total = OU × 1.10 (48场实测校准)
goal_diff = -hcp  (主队让球 → 主队预期净胜)

λ_h = (λ_total + goal_diff) / 2
λ_a = (λ_total - goal_diff) / 2

判型联动:
  D(平局) → λ_total × 0.50
  H/A     → λ_total × 1.10
```
