## 🔍 错误模式分析

### 错误类型分布

| 错误类型 | 场次 | 说明 |
|----------|:---:|------|
| D→H (平局误判为主胜) | 10场 | **核心问题: 模式A过度触发** |
| H→D (主胜误判为平局) | 2场 | 超热门翻车未检测到 |
| D→A (平局误判为客胜) | 3场 | 均势赛客队爆发 |

### 核心问题: 模式A触发过于激进

```
模式A触发条件: 50% < max_imp ≤ 70% + OU ≤ 2.5
26场中19场触发模式A (73%) → 平局预测过多
其中10场实际是主胜 → 误判率53%
```

**典型误判案例**:
- 瑞典 vs 突尼斯 (1.76/3.35/4.70): 隐含H=54%触发模式A → 实际5-1主胜
- 瑞士 vs 波黑 (1.61/3.75/5.00): 隐含H=57%触发模式A → 实际4-1主胜
- 加拿大 vs 卡塔尔 (1.61/3.75/5.00): 隐含H=57%触发模式A → 实际6-0主胜

这些场次的共同特征: **热门方隐含概率54-60%, 实际大胜**。模式A把这个区间一刀切判平局，但实际大部分是主胜。

## 📈 优化建议

### P0: 模式A需要细分

当前模式A把50-70%隐含概率全部判平局，但数据显示:
- 50-55%区间: 平局率高 (~50%) → 判平局合理
- 55-60%区间: 平局率中等 (~35%) → **应判主胜**
- 60-70%区间: 平局率低 (~25%) → 应判主胜

**建议**: 模式A上限从70%收紧到55%，减少误判

### P1: 超热门平局检测

西班牙(82%)和葡萄牙(75%)两场平局未检测到。建议新增模式C:
- imp_H > 72% + 平赔 < 6.0 + OU ≤ 3.0 → 标记"超热门翻车风险"

### P2: 比分预测优化

当前xG从赔率反推，所有平局场景的xG都接近1.2-1.4，导致比分预测集中在1-1。建议引入真实战绩λ(联赛球队可用trend/form API)。
