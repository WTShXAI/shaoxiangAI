# D-Gate 引擎

> 哨响AI核心判型引擎。从v5.2.3纯规则判型演进到v5.2.14多层检测。

## 当前版本: v5.2.14

### 五层平局检测

```
Layer 1:  Mode C — 超热门翻车 (杯赛 imp≥65%)
Layer 1b: Mode C-away — 客队热门翻车 (pa>65%)
Layer 2:  Mode A — 中等热门 (48-70%)
Layer 3:  Mode B — 均衡赛 (spread<0.15, OU≤2.75, od∈[3.0,4.5])
Layer 4:  Default — 窄spread + 全局阈值
Layer 5:  客场爆冷检测 — 浅让盘+低平赔+客胜有空间
```

### 安全阀（按优先级）

1. cs_other < 5.0 → 否决平局
2. od > 7.0 + 单场样本防线漏 → 否决平局
3. 屠杀惯性 (blowout≥33%, total≥2) → Mode C降权
4. 单场样本 + 对手防线漏(ga90≥2.0) → Mode C降权
5. 杯赛防守过滤器 (ga90≥1.0, imp≥72%→×0.85, else→×0.55)

### 关键参数

| 参数 | 值 | 说明 |
|------|-----|------|
| Mode C阈值(杯赛) | 65% | WC小组赛平局率38.5% |
| Mode C阈值(联赛) | 70% | 联赛平局率~25% |
| 防线过滤ga90 | 1.0 | 对手场均失球≥1球触发 |
| 防线降权(高imp) | ×0.85 | imp≥72%时的温和降权 |
| 防线降权(低imp) | ×0.85→×0.55 | imp<72%时的强力降权 |
| COVER_DB球队 | 48 | 小组赛全覆盖 |
