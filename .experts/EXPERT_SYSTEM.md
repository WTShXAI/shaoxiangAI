## 三层闭环

```
回测发现 Bug → 智囊团5人分析 → 产出方案(P0-P3)
    ↓
总工拆解任务 → 路由到专家团执行
    ↓
质检团验收 → GO/NO-GO → 记录员归档
    ↓
回测验证 → 新问题 → 回到智囊团
```

## 各团队详情

### 0. 哨响总工 (shaoxiang-chief)
- 类型: Agent
- 花名: 赵统筹

- 触发: 回测发现问题时激活

### 2. 算法专家团
