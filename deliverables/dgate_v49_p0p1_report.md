## 二、最终落地方案（经回测迭代修正）

### P0-1: 模式A — 2信号协同（放弃上限收紧）

**原方案**：上限 70%→60%
**回测发现问题**：60-70%区间有真实平局（巴西0.68/伊朗0.68/比利时0.68），收紧上限会漏判3场
**最终方案**：保持50-70%区间，但要求**至少2个庄家风险信号协同**

```python
_mode_a_signal_count = sum([_shallow_hcap, _high_water, _highly_balanced,
                             _d_boosted_strong, _extra_draw_signal > 0])
if 0.50 < max_imp <= 0.70 and _low_ou and _mode_a_signal_count >= 2:
    d_gate_active = True; verdict = 'D'
```

**效果**：模式A触发 19→6场，误触发大幅降低

### P1-1: 模式C — 超热门翻车识别

**关键认知**：超热门(imp>72%)平赔应>8.0，<6.0即为庄家异常压低平赔（诱平/防平）。**平赔<6.0本身即庄家信号**，不需额外叠加。


**覆盖场景**：葡萄牙vs民主刚果(od=5.9) ✅ 正确识别

- 函数签名预留：`fifa_rank_diff`, `group_round` 参数

## 六、客观认知

### 规则判型天花板

v4.7→v4.9 准确率持平(42.3%)，印证专家评审判断：**纯赔率结构规则判型的天花板就是42-45%**。

v4.9 的价值不在准确率提升，而在**结构健康化**：
- 模式A误触发大幅降低（19→6场）
- 模式C成功识别超热门翻车（新增能力）
- 主胜/客胜F1显著提升（+0.205/+0.571）
- 平局F1下降是"减少误触发"的合理代价——**宁可不预测也不要误预测**

### 真正的准确率提升路径

规则层价值在**风控标记**（risk_tag 系统），不是主预测。真正提升靠：
- v4.1生产模型（OOF Acc=62.43%）作为主预测
- D-Gate v4.9 作为风控叠加层
- 两者协同：模型判主胜 + D-Gate标记平局风险 → risk_tag 联动比分惩罚
