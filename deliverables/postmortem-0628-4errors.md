## 错误1: 让胜定义bug — "1-0在-1让球下是让平"

### 数学
- hcp=-1.0 (主让1球): handicap score = 主队进球 - 客队进球 - 1
- 1-0 → handicap: 1 - 0 - 1 = 0 → **让平**
- 2-0 → handicap: 2 - 0 - 1 = 1 → 让胜 (赢1分)
- 3-1 → handicap: 3 - 1 - 1 = 1 → 让胜

### 根因
`_score_matches_verdict()` 用 `h >= a` 判让胜, 未考虑让球深度。

### 受影响
- **070 民主刚果**: 判让胜但比分1-0=让平 → 实际3-0确实是让胜, 但标签自相矛盾
- **067 克罗地亚**: hcp=-1.0, 判让胜1-0 → 1-0=让平, 实际2-1=让平

### 修复
```python
def _score_matches_verdict(score_str, verdict, hcp=0):
    h, a = map(int, score_str.split('-'))
    hcp_abs = abs(hcp)
    if '让胜' in verdict:
        if hcp < 0:       return (h - a) > hcp_abs   # 主让: 需净胜>让球深度
        elif hcp > 0:     return (h + hcp) > a        # 主受: 加让球后赢
        else:             return h > a
    if '让平' in verdict:
        if hcp < 0:       return (h - a) == hcp_abs   # 赢球数=让球数
        elif hcp > 0:     return (h + hcp) == a
        else:             return h == a
    if '让负' in verdict:
        if hcp < 0:       return (h - a) < hcp_abs
        elif hcp > 0:     return (h + hcp) < a
        else:             return h < a
```

## 错误2: OU=2.25 trap_low过约束 — 4场全锁小球

### 现象
4场OU=2.25均被强制锚定 ['0-0', '1-1', '1-0'], 总进球≤1。
实际: 2-1(3球), 3-0(3球), 0-0(0球)。仅1/3为小球。

### 根因
trap_low的过滤逻辑 `scores = [s for s in scores if total <= round(exp_goals)+1]` 用 exp_goals=1.0 硬上限, 排除了所有≥2球的比分。

### 机制错误
- 分裂线陷阱是**概率信号**, 不是**确定性屏障**
- 庄家设2.25倾向小球是正确的, 但不应100%锁定≤1球
- 应升权小球比分而非排除大球比分

### 修复方向
- trap_low不再硬排除, 改为降权: 小球比分×1.5, 大球比分×0.5
- 保留0-0/1-1/1-0在锚定但允许2-0/2-1等进入备选
- exp_goals=1.0保持不变用于 `_constrain_ou_to_line` 的市场λ计算

## 错误3: 哥伦比亚0-0 — 一致性校验误剔除正确比分

### 轨迹
1. OU联动锚定: ['0-0', '1-1', '1-0'] — 0-0排第一
2. 策略层: 搏命局→primary='主胜' (哥伦比亚必须赢? 实际上已6分出线)
3. 一致性校验: 0-0不匹配'主胜'→替换为1-0
4. 实际: 0-0

### 根因
**链0战意分析错误**: 哥伦比亚6分已出线, 不应标记为搏命局/主胜。
"已出线→大幅轮换"的信息在战意分析中正确输出, 但策略层仍走了搏命路径。

### 修复
- 战意分析正确输出"哥伦比亚已6分确定出线, 大概率大幅轮换"
- 策略层应使用"已出线=轮换衰减"而非"搏命局"
- 检查context_override逻辑: 6分已出线不应触发must_win_team

## 错误4: 巴拿马0-2英格兰 — 方向偏主队

### 预测链
- 让2球不穿律(R3衰减) → 走水路径
- 名字溢价陷阱 → 小球
- 结果: 0-2 = 让平 (England win by exactly 2)

### 分析
- 名字溢价陷阱判断正确(小球) ✅
- 方向偏巴拿马: 预测让胜(巴拿马covers), 实际让平(走水)
- 0-2 vs 1-1: 均2球, 但方向相反

### 根因
让2球不穿律过度偏向主队方向。当客队是强队时(英格兰实力碾压), 0-2(让平)应比1-1(让胜)更合理。
