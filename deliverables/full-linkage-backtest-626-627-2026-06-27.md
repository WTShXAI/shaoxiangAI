## 三、关键修复：Priority Gate 让2球不穿律

### 问题
6/26 三场让2球比赛（26A/26B/26C）全部触发 Priority Gate "屠杀预警"短路，预测过度屠杀比分（3-3, 0-4, 1-5），但实际都是让2球走水或冷门。

### 根因
`full_linkage_predictor.py` 中 Priority Gate 逻辑：
- 旧：`abs_gap >= 3.0` → 短路到屠杀预警
- 库拉索vs科特迪瓦 abs_gap=3.33（库拉索场均失3.0），但实际0-2走水
- 突尼斯vs荷兰 abs_gap=2.0+阵容不对称，但实际1-3走水

### 修复
新增"让2球不穿律优先"检查（不看gap）：

```python
let2ball_not_penetrate = (sporttery_hcp_abs >= 1.75)  # 竞彩让≥1.75球

if abs_gap >= 3.0 and not let2ball_not_penetrate:
    # 屠杀预警短路 (无让2球约束)
elif abs_gap >= 3.0 and let2ball_not_penetrate:
    # gap≥3 但让2球 → 走水路径优先
elif form_result.massacre_warning and not let2ball_not_penetrate:
    # 屠杀预警短路
elif let2ball_not_penetrate:
    # 走水路径优先 (让胜+让平双选)
```

TaoGe 策略层同步修复：让2球场景输出"让平+让胜"双选，比分锚定走水区（0-1, 1-1, 0-0）。

### 效果
- 让球准确率：50% (3/6) → **66.7% (4/6)**
- 26B 库拉索 vs 科特迪瓦：让负 → 让平 ✅
