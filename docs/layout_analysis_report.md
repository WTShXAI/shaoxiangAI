## 一、目标网站 (zxd6a1.vip) 完整分析

### 1.1 技术栈
- Next.js (React) + Ant Design 4.x + CSS Modules
- react-slick 轮播, Flexbox 布局 (无 Grid)

### 1.2 色彩体系
  主色调: #1890ff (Ant Design 蓝)
  深色导航: #2b3143 (顶部固定导航栏底色)
  功能色: 绿色 #52c41a | 红色 #ff4d4f | 金色 #faad14
  背景: 纯白 #fff 为主, 浅灰 #f5f5f5/#f0f0f0 为卡片底色
  渐变体系 (CTA 按钮/高亮):
    - linear-gradient(90deg, #57eeff, #2595e6)    ← 蓝青色 (主 CTA)
    - linear-gradient(115deg, #50bdff, #1286ff)    ← 深蓝渐变
    - linear-gradient(270deg, #57eeff, #2595e6)    ← 反向蓝青
    - linear-gradient(19.73deg, #44acff, #1883ff, #96ddff)  ← 三色渐变

### 1.3 字体层级
  数字/标题: Akrobat (Bold/ExtraBold/Black) + DIN Pro (DINCondensed-Bold)
  中文正文: PingFang SC, Microsoft YaHei, FZLanTingHeiS-H-GB
  特点: 比分用粗体无衬线字体增强可读性

### 1.5 关键 UI 组件
  - Select 下拉:       赛事筛选
  - Tabs 标签页:       赛前/滚球/赛果切换
  - Card 卡片:         比赛数据容器, hover 阴影上浮
  - Table 表格:        数据列表, 固定表头
  - Drawer 抽屉:       侧边详情
  - Statistic 统计:    比分数值展示
  - Tag 彩色标签:      比赛状态 (进行中=红/未开始=蓝/已结束=灰)
  - Skeleton 骨架屏:   加载态
  - Badge 徽标:        角标提示

### 1.6 体育数据展示模式
  - 比分: 大号 DIN/Akrobat 无衬线字体
  - 赔率: pill 胶囊形状, 颜色编码涨跌
  - 卡片 hover: 阴影浮起 + 蓝色边框高亮
  - 联赛图标: 小圆形/方形 logo

## 二、footballAI 现有前端对比

### 2.1 技术栈对比
  Target:   Next.js + Ant Design + React + CSS Modules
  footballAI: Vue 3 + Element Plus + TypeScript + Vite
  → 差异: 框架不同, 但 UI 库功能对等 (AntD ↔ Element Plus)

### 2.2 布局对比
  Target:   [固定顶导] + [左侧体育导航(固定)] + [中间主区(上:比赛详情,下:投注面板)]
  footballAI: [固定顶导] + [3栏Grid: 左筛选(260px) + 中间比赛卡 + 右工具箱(300px)]

  Target 特征 footballAI 已有:
  ✅ 固定顶部导航
  ✅ 左侧固定侧边栏
  ✅ 中间比赛卡片区
  ✅ 筛选/时间控件

  Target 特征 footballAI 缺少:
  ❌ 纯白背景主题 (footballAI 是深色系)
  ❌ 左侧导航的体育子类图标 + 联赛层级
  ❌ Pill 胶囊形状的概率/赔率展示
  ❌ 蓝青渐变色 CTA 按钮
  ❌ 比赛状态色彩标签 (红/蓝/灰)
  ❌ Drawer 抽屉式详情面板

### 2.3 色彩对比
  Target 亮色系:          footballAI 暗色系:
    #fff 主背景              #0A174E → #0D0D0D 深色渐变
    #1890ff 蓝主色           #00FF9D 霓虹绿主色
    #ff4d4f 红功能色         #FF4757 红功能色
    #faad14 金功能色         #F9C80E 金功能色(平局)
    #52c41a 绿成功色         同

  → footballAI 暗色主题有辨识度, 建议保留, 只引入目标网站的布局模式和组件风格

## 三、具体改造方案

### 改造原则: 保留 footballAI 暗色科技主题, 引入目标网站的布局模式和组件交互

### 3.1 布局结构调整

  当前 (3栏 Grid):                    建议改为:
  +--+--------------+------+          +--+---------------------+------+
  |左| 比赛卡片     | 工具箱|          |左| 比赛卡片网格       | 详情 |
  |侧| +----------+ | +---+ |          |侧| +-----+ +-----+   | 抽屉 |
  |栏| | Card 1   | | |工具| |          |导| |Card1| |Card2|   | 面板 |
  |  | | Card 2   | | | 1  | |          |航| +-----+ +-----+   | (按需 |
  |  | | Card 3   | | |工具| |          |  | |Card3| |Card4|   |  弹出)|
  |  | | ...      | | | 2  | |          |  | +-----+-----+   |      |
  +--+--------------+------+          +--+---------------------+------+
                                       | 底部: 图表/统计面板           |
                                       +-----------------------------+

  改为 2栏: 左侧联赛导航 (220px) + 右侧自适应主区
  工具箱改为 Drawer 抽屉 (点击卡片上的分析按钮滑出)

### 3.2 左侧联赛导航 (仿目标网站)

  当前 FilterSidebar (筛选器面板) → 改为 联赛导航面板:
  ```
  +------------------+
  | [全部联赛]       |  ← 默认选中
  | [热门联赛]       |
  | ⚽ 英超          |  ← 联赛logo + 文字 + 比赛数badge

  | [更多联赛...]    |
  +------------------+
  ```

### 3.3 比赛卡片 (仿目标 pill badge 风格)

  当前 MatchCard 保留结构, 增强风格:

  ```
  ┌──────────────────────────────┐
  │ [英超] 21:00  进行中 45'     │  ← 联赛 + 时间 + 状态tag (红/蓝/灰)
  │                              │
  │ 曼联  [2]  -  [1]  利物浦    │  ← 队名 + 大字比分 (DIN/Akrobat feel)
  │                              │
  │ 预测: 主胜 ████████░░ 72%   │  ← 概率条 + pill badge 百分比
  │       ┌─┐ ┌─┐ ┌─┐           │
  │ 推荐: │H│ │D│ │A│           │  ← 胶囊按钮, 推荐项蓝渐变高亮
  │       └─┘ └─┘ └─┘           │
  │                              │
  │ 预测比分: 2:1  1:0  1:1     │
  │ 置信度: [████████░░] 85%    │
  │                              │
  │ [详细分析 →] [让球 →] [大小球]│  ← 操作按钮
  └──────────────────────────────┘
  ```

### 3.4 CTA 元素配色更新

  保留足球AI的霓虹绿 (#00FF9D) 作为主色调, 但引入蓝渐变过渡:

  - 主要按钮:   linear-gradient(90deg, #00FF9D, #00CC7A)  ← 当前绿渐变
  - 次级按钮:   linear-gradient(90deg, #1D267D, #0A174E)  ← 深蓝渐变 (仿目标)
  - 高亮标签:   linear-gradient(90deg, #57eeff, #2595e6)  ← 蓝青渐变 (仿目标, 用于"专家推荐"等)
  - 概率条:     #00FF9D 填充, #1A1A2E 底色

### 3.5 新增交互组件

  (1) Drawer 抽屉面板 → 替代右侧 ModelToolboxSidebar
     - 点击卡片"详细分析" → 右侧滑出分析面板
     - 内含: 比分预测/让球/大小球/冷门检测 tabs
     - 仿 Ant Design 风格, 半透明遮罩 + 滑入动画

  (2) Tabs 标签页 → 比赛类型切换
     - [全部比赛] [今日] [滚球] [赛果]
     - 替代当前的 TimeFilterBar

  (3) 比赛状态 Tag 彩色标签
     - SCHEDULED → 蓝色 #3498DB pill
     - LIVE → 红色 #FF4757 pill (带脉冲动画)
     - FINISHED → 灰色 #6B6B80 pill

  (4) Pill Badge 概率展示
     - 当前是百分比数字 → 改为胶囊形状 + 颜色深度表示置信度
